Source files use UTF-8 text and the extension .ssk.

Programs

A program is a sequence of Sanskrit sentences. Each sentence ends with . or ।.

gaṇakaḥ pañca phale nidadhāti.
gaṇakaḥ phalaṃ dvābhyāṃ vardhayati.
gaṇakaḥ phalaṃ darśayati.

Sentences

Each executable sentence currently has one finite verb and the participants required by that verb frame.

FrameRequired rolesEffect
nidadhātikarman + adhikaraṇastore a value in a location

The sentence profiler also recognizes questions, relative phrases, coordinated clauses, and verbless profiles as grammar objects.

Nouns

The current subanta engine generates selected a-stem masculine, a-stem neuter, and ā-stem feminine forms.

devaḥ, devam, devena, devāya, devāt, devasya, deve
phala, phalaṃ, phalena, phalāt, phalasya, phale, phalāni
latā, latām, latayā, latāyai, latāyāḥ, latāyām

Verbs

The tiṅanta engine currently models present, command, and potential forms for selected stems.

lakāraSenseExample
laṭpresentbhavati
loṭcommandbhavatu
vidhiliṅpotentialbhavet

Both parasmaipada and ātmanepada are represented in morphology, but only selected forms are executable today.

Numbers

Small cardinal numerals from 0 through 10 are registered in object and instrumental roles.

pañca phale nidadhāti.
phalaṃ dvābhyāṃ vardhayati.

Pronouns

Basic first, second, and third person forms are recognized by morphology.

aham, āvām, vayam
tvam, yuvām, yūyam
saḥ, tat, tam, tena

Upasarga

The controlled prefix registry includes pra, parā, apa, sam, anu, ava, nis, dus, vi, ā, ni, adhi, ati, su, ud, abhi, prati, pari, and upa.

Scripts

IAST is the canonical development input. The transliteration module supports current-form IAST and Devanagari conversion.

gaṇakaḥ -> गणकः
phalaṃ -> फलं
nyūnayati -> न्यूनयति

Sandhi

First-pass sandhi support covers selected vowel and visarga joins.

deva + atra -> devātra
deva + iti -> deveti
rāmaḥ + atra -> rāmotra

Limits

The current language is not full Sanskrit yet. Every canon item now has at least partial treatment, but the canon gate blocks release until those partial batches become fully implemented behavior.
